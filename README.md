
# HematoVision – Advanced Blood Cell Classification Using Transfer Learning

## Project Overview
HematoVision is an AI-powered system designed to automatically classify microscopic blood cell images using Transfer Learning and Convolutional Neural Networks (CNNs). It helps healthcare professionals analyze blood samples efficiently and accurately.

## Classes
- Eosinophil
- Lymphocyte
- Monocyte
- Neutrophil

## Features
- Upload blood cell images
- Automatic classification
- High accuracy using transfer learning
- Dataset management
- Educational and healthcare support

## Technologies
Python, TensorFlow, Keras, OpenCV, NumPy, Matplotlib, Scikit-learn
