# HematoVision – Advanced Blood Cell Classification Using Transfer Learning

## 📌 Project Overview
HematoVision is an AI-based medical image classification system designed to automatically classify blood cells using Transfer Learning and Convolutional Neural Networks (CNNs). The system analyzes microscopic blood smear images and classifies them into:

- Eosinophils
- Lymphocytes
- Monocytes
- Neutrophils

This solution reduces manual workload for pathologists and improves diagnostic speed and accuracy.

---

## 🎯 Problem Statement
Manual blood cell classification is time-consuming, requires expert knowledge, and is prone to human error. There is a need for an automated, scalable, and accurate system that can classify blood cells efficiently.

HematoVision provides an intelligent solution using deep learning and transfer learning.

---

## 🎯 Objectives

## 📂 Dataset

Dataset contains:

- 12,000 annotated blood cell images
- Classes:
  - Eosinophil
  - Lymphocyte
  - Monocyte
  - Neutrophil

Image size: 224x224 pixels
